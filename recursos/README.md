# 📚 Recursos para Aprender Python

Esta pasta existe para **orientar**, não para entregar tudo pronto.

Aqui está o **ponto de partida mais importante** para quem está aprendendo Python:
a documentação oficial.

Aprender a **consultar documentação** faz parte do caminho de todo programador.

---

## 🐍 Documentação Oficial do Python

A documentação oficial é a **fonte mais confiável** sobre Python.

👉 https://docs.python.org/pt-br/3/

Use para:
- Entender como a linguagem funciona
- Consultar tipos, funções e módulos
- Tirar dúvidas reais enquanto pratica
- Criar o hábito de pesquisar por conta própria

Não é necessário ler tudo.
Use como referência sempre que algo não fizer sentido.

## 🧠 Conselhos importantes para quem está aprendendo

Pesquisar faz parte do aprendizado.

Quando surgir uma dúvida:
1. Leia o erro com calma  
2. Consulte a documentação  
3. Teste no código  
4. Ajuste e continue  

Não existe atalho sólido para aprender programação.

Este repositório não quer te deixar dependente.  
Quer te tornar **autônomo**.

---

Não tente aprender tudo ao mesmo tempo.  
Não se compare com quem está mais avançado.

Todo desenvolvedor já foi iniciante um dia.

Se travar, volte um passo.  
Se errar, continue.  
Se cansar, descanse — **não desista**.

Este repositório existe para isso.

---

> Se você aprender a usar a documentação,  
> você aprende qualquer linguagem no futuro.
